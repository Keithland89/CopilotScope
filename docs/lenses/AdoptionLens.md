# AdoptionLens

*The core "are people actually using it?" lens — usage, reach, and activation.*

## What it answers

- Who is **active** on Copilot, and how many of the enabled population is that?
- How **widely** has Copilot spread across teams, apps, and surfaces?
- How does engagement **trend** over time (activation, retention, reach)?

## Source tables

| Table | Role | Required? |
|---|---|---|
| Audit log | Copilot usage events | **Core** |

## Key measures

- **Active users** / **Enabled users** — the population actually using Copilot vs entitled to.
- **% Active** — activation rate.
- **Reach** — spread across apps / surfaces / teams.

## Where it lives (pages)

- **Activation**
- **Adoption & Reach**
- **Heatmap Trend**
