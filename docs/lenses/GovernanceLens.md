# GovernanceLens

*Agent health, governance, and feedback — a first-class governance view for admins and CoE teams.*

## What it answers

- What **agents** exist across the estate, and how **healthy** is adoption of each?
- What **governance** signals need attention (ownership, sprawl, risk)?
- What is **user sentiment** from product feedback?

## Source tables

| Table | Role | Required? |
|---|---|---|
| Agent tables | Agent inventory & health signals | **Required for this lens** |
| Product feedback / Agent 365 | Sentiment & governance signals | Required for the feedback views |

## Key measures

- **Governance / health** — agent inventory, adoption health, risk signals.
- **Sentiment** — feedback rolled up across agents and the estate.

## Where it lives (pages)

- **Agent Governance**
- **User Feedback**

## Setup

GovernanceLens lights up when the **product feedback / Agent 365** source is loaded. See
